## 🆘 TROUBLESHOOTING

**"pip install fails"**
→ Use `pip install -r requirements.txt --break-system-packages`

**"ModuleNotFoundError"**
→ Ensure you're in the virtual environment: `source venv/bin/activate`

**"streamlit not found"**
→ Install separately: `pip install streamlit`

**"Churn model fails to load"**
→ Ensure models/ directory exists with .pkl files (should be there)

**"For more help"**
→ Check README.md "Troubleshooting" section or PROJECT_VALIDATION.py output
